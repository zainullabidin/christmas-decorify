# Christmas Decorify

A Christmas-themed static website I designed in Figma and built with HTML and CSS. It's a mock e-commerce site for Christmas decorations with a product catalog, a quiz that recommends items based on your answers, and an about us page.

---

## Pages

**Homepage** - Landing page with a banner, featured products, decoration categories (indoor, outdoor, tree decorations), and a quiz recommendation section.

**Products** - Full product listing with ratings, prices, and category filters for tree decorations, indoor items, and outdoor decor.

**Quiz Recommendation** - A short quiz that helps you pick decorations based on what you're looking for. Answer a few questions and it suggests products.

**About Us** - Background on the project, how the site works, what makes it different, and a section about the team.

---

## Built With

- HTML5
- CSS3
- Google Fonts (Mountains of Christmas, Quicksand)
- Figma (full design mockups done before coding)

## Design

The full Figma design includes 8 pages (Homepage, Products, Product Detail, Cart, About Us, Contact Us, Login/Signup, Checkout). The 4 core pages are implemented in code. The complete design can be viewed in the Figma file.
